🚗 AI帰り道ナビ：中部地方限定の寄り道サポートアプリ
「AI帰り道ナビ」は、学校や仕事の帰り道に「どこか寄りたいな」と思った時、あなたのルート沿いにある最適なスポットをAIが提案してくれるアプリです。

✨ 主な特徴
中部地方専用: 愛知・岐阜・三重・静岡など、中部エリアの情報に特化しています。

ルート沿い検索: 出発地と目的地を入力するだけで、移動経路から大きく外れない便利なスポットを見つけ出します。

AIによる柔軟な検索: 「スタバ」などの略称や、「ゆっくり勉強したい」「お腹空いた」といった抽象的な要望にもAIが応えます。

📖 使いかたガイド
1. ルートを入力する
画面上部の入力欄に情報を入れましょう。

出発地・目的地: 「名古屋駅」「栄」など、あなたの移動ルートを入力します。

※片方だけ入力した場合は、その周辺のスポットを検索します。

何して遊ぶ？: 「カフェ」「ゲーセン」「静かな場所」など、今の気分を入力してください。

2. ワンタップでかんたん検索
入力が面倒な時は、検索窓の下にあるアイコン（🎢 テーマパーク、☕ カフェ、🎮 ゲーセンなど）をタップするだけで、すぐに検索が始まります。

3. スポットを確認する
検索結果は地図上にピンで表示され、右側のパネルにはスポットの詳細リストが表示されます。

気になるお店をリストでクリックすると、地図がその場所にズームします。

AIからの「寄り道アドバイス」も表示されるので、参考にしてみてください。
